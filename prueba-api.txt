
ACTIVIDA INFORME DE PRUEBA DE API (POST /REGISTRO)


1. DETALLES DE LA PETICIÓN EN THUNDER CLIENT:

- Dirección: http://localhost:3000/registro
- Método HTTP: POST
- Formato del cuerpo: JSON (application/json)
- Datos enviados:
  {
    "nombre": "Maria",
    "mensaje": "Hola comunidad"
  }


2. RESPUESTA EXACTA DEVUELTA POR EL SERVIDOR:

Status Code: 200 OK

{
  "estado": "Datos recibidos",
  "nombre": "Maria",
  "mensaje": "Hola comunidad"
}


3. EXPLICACIÓN DETALLADA DE LA RESPUESTA:

Al realizar de forma correcta la petición POST utilizando la pestaña 
Body en Thunder Client, el servidor Express procesa la información
exitosamente y devuelve un objeto JSON junto con un código
de estado HTTP 200 OK.
